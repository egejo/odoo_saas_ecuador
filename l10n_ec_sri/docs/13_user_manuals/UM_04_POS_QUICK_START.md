# USER MANUAL: POS QUICK START
## UM_04 - Guía Rápida Punto de Venta

**Document ID**: UM-004 | **Version**: 1.0 | **Date**: 2026-01-22
**Owner**: Training Lead | **Audience**: Cashiers

---

## 1. ABRIR CAJA

1. Ir a **Punto de Venta**
2. Clic en su **POS asignado**
3. Ingresar **saldo inicial de caja**
4. Clic **"Abrir Sesión"**

---

## 2. REALIZAR VENTA

### Agregar Productos
- **Escanear código de barras**, o
- **Buscar por nombre**, o
- **Seleccionar categoría**

### Identificar Cliente

| Monto | Acción |
|:------|:-------|
| ≤ $50 | Consumidor Final (automático) |
| > $50 | **Obligatorio**: Ingresar RUC/Cédula |

### Cobrar

1. Clic **"Pago"**
2. Seleccionar método: Efectivo / Tarjeta / Transferencia
3. Ingresar monto recibido
4. Sistema calcula **cambio**
5. Clic **"Validar"**
6. Se imprime **RIDE automáticamente**

---

## 3. CERRAR CAJA

1. Clic **"Cerrar Sesión"**
2. Contar efectivo físico
3. Ingresar monto contado
4. Sistema compara con esperado
5. Si hay diferencia → Reportar
6. Supervisor aprueba cierre
7. Se genera **Reporte Z**

---

## 4. DEVOLUCIONES

1. Buscar venta original
2. Clic **"Devolver"**
3. Seleccionar productos
4. Genera Nota de Crédito
5. Reembolsar al cliente

---

## 5. PROBLEMAS COMUNES

| Problema | Solución |
|:---------|:---------|
| "Sin conexión" | Modo offline activo, sincroniza después |
| "Total > $50 sin ID" | Solicitar cédula/RUC al cliente |
| "Producto no encontrado" | Verificar código, buscar por nombre |
